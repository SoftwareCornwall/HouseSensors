# :sparkles: House_Sensors :sparkles: 

Contains source code for WaterSensor, HumiditySensor and a server/front end for displaying the data.

## Dependencies


### For HumiditySensor:

* RTIMULib - librtimulib-dev
* libcurl - libcurl4-openssl-dev:armhf


### For WaterSensor:

* wiringPi - wiringpi
* libcurl - libcurl4-openssl-dev:armhf
